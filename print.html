微信通知功能设置说明

微信通知由第三方提供，可能存在延迟通知或偶尔丢失消息等情况。 微信通知消息有发送次数限制，具体限制规则由微信官方限制，但是日常使用基本没有问题。

步骤分三步

  • 1、关注公众号，并获取专属推送地址
  • 2、关闭这个公众号的消息免打扰
  • 3、粘贴专属地址到 鱼缸卫士 app

使用方法

获取第三方通知的webhook地址

网上有很多提供第三方微信通知的服务，这里以showdoc推送服务为例子

showdoc推送服务 是东莞市冬惊鱼公司提供的一个微信推送服务。

简易教程

获取专属地址

微信搜索关注公众号 showdoc推送服务 -> 关注 -> 发送任意消息给它 -> 长按它回复的专属地址 -> 复制

打开公众号消息提醒

微信打开公众号showdoc推送服务 -> 点右上角 小人图标查看资料 -> 再点右上角三个点 ··· 图标 -> 设置 -> 关闭 消息免打扰

详细说明

打开您的手机微信，扫码下面二维码 ，或者使用微信搜索 showdoc推送服务 关注公众号： showdoc推送服务

关注成功后，如果这个公众号没有给你发送 推送地址。

您可以使用微信给它发送一个消息，内容随意，它会发送 一个专属推送地址给你。复制这个地址

格式类似 https://push.showdoc.com.cn/server/api/push/***************
